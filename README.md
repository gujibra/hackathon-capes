## Objetivo

Projeto feito para o hackathon da CAPES


## Execução 

1. ApiKey:
Crie uma conta no Groq.com para conseguir uma api key e coloque ela na config.json

2. Instale as dependências:

```sh
npm install
```

4. Execute o aplicativo:

```sh
npm run dev ou 

npm run build  
npm run start
```
